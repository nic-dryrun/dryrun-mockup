PROPOSED MOCKUP Code Security Intelligence — Insights Redesign
Proposed redesign of the Insights page. Renamed to Code Security Intelligence. Replaces the open-ended AI chat with structured, click-to-run security workflows that produce templated reports. The AI assistant is retained for ad-hoc queries. Existing capabilities (search, summarize, analyzer stats, file history) are preserved and power the workflows behind the scenes.
Current: Insights
Open-ended AI chat — users must know what to ask
  • Pre-built question pills (generic)
  • Custom categories with free-text prompts
  • Daily auto-run results as a flat list
  • No structured output format
  • Users guess what workflows are possible
Proposed: Code Security Intelligence
Structured, click-to-run workflows — users see what's possible
  • 10 named workflows with clear descriptions
  • Each produces a templated document/report
  • Scheduled workflows with configurable cadence
  • Report library with history and versioning
  • AI assistant retained for ad-hoc exploration
