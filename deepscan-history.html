Business Purpose

API-only Ruby on Rails service ("Simple API") exposing REST endpoints for security scanning and code analysis data. The root path redirects to Swagger UI at /api-docs. It provides two namespaces: a /v1 external-facing REST API for customers to automate access to security scanning data and configuration, and an /internal API for internal dashboard features using JWT tokens minted by dryrun-api.

Target Audience

External customers/integrators using API keys to programmatically interact with security scanning data and configurations via /v1. Internal dashboard users authenticated via JWT tokens to access /internal endpoints.

Data Handling

  • Data types handled: Security scan results, vulnerability findings, repository metadata, user account information, API keys, JWT tokens
  • Storage: PostgreSQL (primary), Redis (sessions/cache)
  • Sensitive data flows: API keys transmitted in headers, JWT tokens in auth cookies, scan results contain code snippets

Tech Stack

  • Ruby on Rails 7.1, Ruby 3.2
  • PostgreSQL 15, Redis 7
  • Puma web server, Sidekiq background jobs
  • Deployed on Kubernetes (GKE)
